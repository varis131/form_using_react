# React Form Basics

A fully controlled React form built for practice, demonstrating clean form
state management and input handling.

## Features
- Controlled form inputs
- Single `useState` object for form data
- Text, email, select (dropdown) inputs
- Checkbox handling
- Form submission with console output
- Clean and simple CSS styling

## Concepts Covered
- Controlled Components
- Object-based state management
- `onChange` and `onSubmit` handling
- Dropdown (`select`) and checkbox inputs
- Basic form structure in React

## Tech Stack
- React
- JavaScript
- CSS

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Learning Goal
This project was created to strengthen React fundamentals related to form
handling and controlled components.
